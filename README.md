# 🚀 JobGenie AI

> An AI-powered career assistant that generates personalized interview preparation plans and ATS-friendly resumes tailored to your target job description.

---

## 🌐 Live Demo

- **Frontend:** https://job-genie-ai-theta.vercel.app
- **Backend:** https://jobgenie-ai-backend.onrender.com

---

## ✨ Features

### 🔐 Authentication

- User Registration
- Secure Login
- JWT Authentication
- HTTP-Only Cookie Authentication
- Protected Routes
- Logout Functionality

### 🤖 AI Interview Plan Generator

- 📄 Upload Resume (PDF/DOCX)
- 📝 Enter Self Description
- 💼 Paste Job Description
- 🎯 AI Match Score
- 💻 Technical Interview Questions
- 🗣️ Behavioral Interview Questions
- 📊 Skill Gap Analysis
- 📅 Personalized Preparation Plan

### 📄 AI Resume Generator

- ATS-Friendly Resume Generation
- Resume Tailored to Selected Job
- Professional Resume Formatting
- PDF Generation
- One-Click Download

---

## 🛠️ Tech Stack

### Frontend

- React.js
- React Router
- SCSS
- Axios
- Context API

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cookie Parser

### AI & PDF

- Google Gemini API
- Puppeteer
